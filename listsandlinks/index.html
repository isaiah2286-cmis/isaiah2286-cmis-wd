Creating a New Webpage in the Production Server

  1. Type "gedit file name" in terminal
  2. Copy and paste everything in template.html into the new webpage
  3. In the <title> part, type in the title of your webpage
  4. For the body you can have header (<h1>) and main paragraph (<p>)
  5. Remember to put </h1> and </p> at the end of each element
  6. After you are done with the webpage press Ctrl+S and switch back to your terminal
  7. Tpye "sftp name####@cmis-hsteacher" in the terminal and enter the password, then type "cd public_html/"
  8. Type "put file name" to upload your webpage
  9. To view your webpage, you can type in "http://cmis-hsteacher/~name####"
  10. Congratulation, you have created a webpage on the production server
