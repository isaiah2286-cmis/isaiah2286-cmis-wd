Press start and hit space bar twice at the same time
Type terminal
Type 'git clone --config="http.proxy=http://your-network-username:your-network-password@192.168.2.1:8080" https://github.com/your-github-username/your-github-repository-name.git' in you terminal and press enter
Type cd 'https://github.com/your-github-username/your-github-repository-name'and press enter
Type 'gedit the file name you want'and press enter
Type whatever you want in gedit
Press Ctrl+S to save then close it
Type 'git add the file name you want' and press enter
Type 'git commit -m "whatever commit message"' and press enter
Type 'git push' and press enter
Open you github account and find the stuff you entered
Great! You are done
